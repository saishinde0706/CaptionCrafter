********CaptionCrafter** is an AI-powered image captioning tool that generates creative, descriptive, and context-aware captions for any image you upload.  
Built using **Python**, **OpenAI’s API**, **Gradio**, and standard image processing libraries, it provides a clean and interactive interface to bring your pictures to life with words.

---

## Features
- **AI Caption Generation** — Uses OpenAI models to analyze and describe image content.  
- **Upload Interface** — Simple drag-and-drop image upload via Gradio.  
- **Easy Setup** — Just a few lines to install and run.  
- **Customizable Output** — You can modify prompts to control tone or style (funny, poetic, simple, etc.).

---

## Tech Stack
- **Python 3.8+**
- **OpenAI API**
- **Gradio** (for the web interface)
- **Pillow / Image libraries**
